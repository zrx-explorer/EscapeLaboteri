# 《逃离来伯特利》策划案 · 修订版 v2.0

> 本文件为对原始《逃离来伯特利策划.docx》的迭代修订。
> 修订原则：补齐核心循环 / 完善内奸维度 / 平衡职业数值 / 增加元进展 / 明确网络与运营。

---

## 第 0 章 · 体验目标（MDA）

| 维度 | 设定 |
|---|---|
| **Aesthetics** | 幻想（dungeon-crawl）、紧张（毒圈）、猜疑（社交推理）、反转（PvP 第 4 关） |
| **Dynamics** | 5 人合作中暗藏 1 人对抗，前 2 关合作但内奸暗中破坏，第 3 关 Boss 决战，第 4 关全面火并 |
| **Mechanics** | 动作 ARPG + 隐身/陷害/伪装 + 商店 + 投票 + 元进展 |

**核心情绪曲线**：

```
情绪强度
  ▲
  │            ┌── PvP 角斗 ──┐
  │      Boss┐           │
  │       │              │
  │  讨论│  讨论│
  │ 关1  关2  关3
  └─────────────────────────────► 时间
```

## 第 1 章 · 核心循环（Core Loop）

```
[选英雄] → [关1: 合作PvE] → [章节末: 揭晓内奸 + 30s讨论 + 匿名投票]
        → [商店: 100g装备] → [关2: 三选一环境] → [商店: 200g装备+奸商]
        → [关3: Boss战(行商紧急补给)] → [是否触发PvP第4关]
        → [结算] → [元进展: 圣杯天赋] → 下一局
```

**预期时长**：18~22 分钟/局。

## 第 2 章 · 玩家与身份

### 2.1 配置

| 玩家数 | 内奸数 | 备注 |
|---|---|---|
| 1~3 | 0 | 训练 / PvE 模式 |
| 4 | 1 | 新增 |
| 5 | 1 | |
| 6 | 1 | 新增 |
| 7 | 2 | 新增 |
| 8 | 2 | 内奸互知 |

### 2.2 内奸"亮明"机制（新增量化）

- 主动按 `R` 亮明，**不可逆**。
- 亮明后获得：
  - ATK +15%，移速 +10%
  - 被动【血色狂热】：每 10s 减 1s 技能冷却
  - 解锁专属"夺魂"普攻特效

### 2.3 死亡

- 自动转幽灵观战，可发布**1 条 ≤12 字遗言**（避免直接报点）。
- 死亡身份**结算时统一揭晓**，避免遗言泄密。
- 第 3 局以上：解锁"残存任务"（幽灵开宝箱给队友送金币）。

## 第 3 章 · 职业与数值（重新平衡）

### 3.1 数值定位修复（解决"猎人 ATK 高于术士"问题）

| 职业 | 1-2级 HP | ATK | 攻速 | 射程 | 修订点 |
|---|---|---|---|---|---|
| 骑士 | 1000 | 55 | 1.0 | 1m | — |
| 狂战士 | 800 | 62 | 1.1 | 1m | — |
| 猎人 | 600 | 80 | 1.2 | **6m** | 射程 5→6m，强化 kite |
| 圣骑士 | 800 | 60 | 1.0 | 1m | — |
| 术士 | 700 | **50** | 1.0 | 5m | ATK 65→50，技能伤害 +30% 补偿 |

> 术士被打回控制定位，与猎人区分明确。

### 3.2 嘲讽公式（量化）

```
仇恨值 = (5 - 嘲讽等级) × 100 + 1000 / max(距离, 0.5)
敌人优先攻击仇恨值最高的目标。
```

### 3.3 Boss 减弱（量化）

- 每 1 名内奸死亡：Boss 当前 HP −15%、ATK −10%。
- 不会触发狂暴，仅净削弱。

## 第 5 章 · 关卡设计补丁

### 5.1 第 0 关：教学（新增）

- 时长 2 min，强制新人。
- 目标：操作 → 普攻 → 技能 → 商店 → 内奸识别课。

### 5.2 关卡 2 三选一深化（解决"差异不足"）

| 子关卡 | 玩法 verb（新增） |
|---|---|
| 熔岩狱 | "灭火"：踩水池熄灭岩浆地砖 |
| 水牢 | "憋气"：氧气槽，浮出水面回 ox |
| 魔法陷阱 | "拆除"：3 阶 QTE 拆魔法塔 |

### 5.3 第 4 关 PvP 改判定（解决"平局不爽"）

- 3 min 强制结束 → 改为：3 min 后地图开始 **每 5s 缩 10%**，圈外受 50 真伤/秒，**强制分胜负**。

## 第 6 章 · 沟通与投票（新增）

### 6.1 信号轮盘

按 `V` 弹出 8 方向轮盘：

```
        小心↑
   救我↖   ↗去那边
集合←        →有商人
   攻击↙   ↘他可疑
        撤退↓
```

### 6.2 章节末投票

- 30s 讨论 → 匿名投票 → 票数最多者放逐（失去 1 关行动力，不死亡）。
- **平票不放逐**。
- 内奸亮明后**不能被投票**（防止 1 关秒删内奸）。

## 第 7 章 · 经济与元进展（新增）

### 7.1 单局经济（修复溢出）

- 通关结束如金币 > 500，**奸商赌博**：100g 抽 1 次水晶/装备。

### 7.2 元进展 · 圣杯系统

- 每局根据表现获得 **5~30 圣杯碎片**。
- 5 类天赋树：
  - **战意**：基础属性 +1%~10%
  - **智略**：技能冷却 -1%~10%
  - **谋略**：开局 +金币 / 道具
  - **传承**：皮肤、表情解锁
  - **异端**：仅在被分配为内奸时生效（额外携带 1 内奸技能）

### 7.3 通行证（运营层）

- 4 周一赛季，60 级通行证。
- 免费道：每 5 级送水晶/经验书。
- 高级道：每级送皮肤碎片。

## 第 8 章 · UI / UX 规范

- 屏幕安全区：1080p 下 5% 边距。
- HP/MP 浮动：受击向上跳字，治疗绿色，伤害红色。
- 暗色基调，金色高亮（古迹幻想）。
- **小地图**：内奸视野中显示同伴金色光环，普通玩家不显示。

## 第 9 章 · 网络架构（新增）

### 9.1 拓扑

```
客户端 ◄── 状态同步(20Hz) ── 房间服务器(Node.js + Colyseus)
        ── 操作上行(60Hz) ─►
```

### 9.2 权威与防作弊

- 服务器权威：所有伤害、命中、技能在服务器结算。
- 客户端预测 + 回滚（用于移动平滑）。
- 反外挂：行为指纹（按键间隔聚类） + 帧 hash 校验。

### 9.3 房间生命周期

- 创建房（房主）→ 等待（30s 招人）→ 开局 → 单局结束销毁
- 断线 5s 内重连：恢复状态；超时托管 AI。

## 第 10 章 · KPI 与平衡

| 指标 | 目标 |
|---|---|
| 平均局时长 | 18~22 min |
| 平均关卡死亡数 | 1.0 ± 0.3 |
| 内奸胜率 | 45%~52%（高于 50% 不可接受） |
| 玩家次留 | ≥ 35% |
| 7 日 ARPU | ≥ 0.8 元 |

平衡测试方法：每个版本采集 1k+ 局数据，按英雄/内奸/玩家数分档统计。
